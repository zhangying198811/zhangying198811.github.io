Mysql远程连接授权

Mar 12, 2018 | 技术

连接数据库报错，root无外网授权，现在授权某一个IP的访问

> create user ‘coursemanager2’@’%’ identified by ‘881121abC&*’;

> grant select, insert, update, delete, create, alter, event, drop, references, create temporary tables, create view, show view, alter routine, create routine, execute, index on . to ‘coursemanager2’@’%’ identified by ‘881121abC&*’;

> select * from mysql.user\G;

还是报错，不行，原来是mysql只授权了本机ip

将这个地址改为0.0.0.0

大功告成，外网可访问
