1、安装Git、GitHub配置

step 1.安装Git

前往Git官网下载Windows版本压缩包，下载完成后解压安装。

https://git-scm.com/downloads/

下载git安装文件，双击执行安装。一直下一步即可。

step 2.注册Github账户

前往Github网站，注册一个新用户。用邮箱注册的一定前往邮箱去验证邮件。要不然之后可能会有小问题。

然后在github-bash中输入命令：ssh -T git@github.com

然后刷新网页会发现SSH-key前面的钥匙变成了绿色，说明SSH-key配置成功

step4. Github上创建一个新的repository

在自己的Github主页右下角,创建一个新的repository。比如我的Github用户名为zhangying8811，那么我创建的repository的名字应该是 zhangying8811.github.io 。

这里严重注意:

一定要以你的Github用户名.github.io创建。假如我没有用zhangying8811.github.io而是用了a.github.io，那么当我浏览器访问博客的时候会出现404错误。这里并不是没有部署成功，而是把它部署在了这里:http://zhangying8811.github.io/a.github.io。所以，如果想直接zhangying8811.github.io访问，那么就需要和用户名保持一致。

注意：

  1. 有些用户没有设置Github的SSH，会导致上面两句失败。SSH的介绍和设置方法可以查看官方教程,配置起来很简单。如果英文看不明白或者过程中出现小问题，可以查看我写的SSH设置教程，是对官方教程的解释和扩展，针对配置过程中的小问题都有解决办法。
