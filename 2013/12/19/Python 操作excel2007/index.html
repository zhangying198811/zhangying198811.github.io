先介绍下excel2007的python 插件openpyxl的安装及使用方式（windows Win7）

1、安装openpyxl

​ 1)首先安装easy_install.

从官网下载一个包解压后用python setup.py install
此时会报错：No module named setuptools.需要先下载ez_setup.py在运行python.exe ez_setup.py 并将D:\Program Files\python2.7\scripts添加到环境变量中

2)安装openpyxl

打开openpyxl文件夹执行python setup.py bdist_egg 然后执行python setup.py bdist_egg 最后执行python setup.py install

安装完成。
